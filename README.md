# indiasmedicinebank
💊 India’s Medicine Bank (IMB)

📌 Project Overview

India’s Medicine Bank (IMB) is a web-based platform that connects medicine donors and receivers.
It helps reduce medicine wastage and ensures that unused medicines reach people in need.

---

🚀 Features

- 👤 User Registration & Login
- 🔐 OTP Verification System (Generated on Website)
- 💊 Donate Medicines
- 📥 Request Medicines
- 📊 Admin Dashboard
- 🔄 Real-time Request Handling
- 📍 Location-based Filtering
- 🗂️ Medicine Management System

---

🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Python (Flask)
- Database: SQLite
- API: RESTful API Integration
- Authentication: OTP-based Verification

---

⚙️ System Design Concepts Used

- 🧠 Operating System Concepts
  
  - Process handling (user requests)
  - Resource management (medicine allocation)

- 📊 Algorithm Analysis
  
  - Efficient request matching
  - Data filtering and searching

🔧 Installation & Setup

1. Clone Repository

2. Navigate to Project

3. Install Dependencies

pip install flask

4. Run Application

python app.py

5. Open in Browser

http://127.0.0.1:5000/
